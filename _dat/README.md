# Datasets

## *ICU-Chronic*, *ICU-Acute*, and *ICU-Mixed*
Since accessing the datasets including *ICU-Chronic*, *ICU-Acute*, and *ICU-Mixed*, requires authorization, we provide instructions for the installation.


1. Download mimic3-benchmarks from https://github.com/YerevaNN/mimic3-benchmarks into the mimic3-benchmarks-wrapper_time-to-event directory.

2. Run preprocss.sh \
    `sh preprocess.sh`
